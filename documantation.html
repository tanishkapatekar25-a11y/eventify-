Project Documentation

Structure, features, and technologies used in Eventify

Project Structure

index.html handles the landing page, categories.html lists all events, and the events/ folder contains 15 dedicated event pages.

styles.css centralizes layout, colors, responsive behavior, and animations. script.js manages interactivity, while events.js stores reusable event data.

Main Features

  • Hero slideshow and category quick links on the homepage
  • Search and filter tools for event discovery
  • Countdown timers for the upcoming event and individual event pages
  • Dark/light toggle plus teal/neutral theme switcher
  • Popup confirmation and ticket badge logic for forms
  • Dashboard, FAQ, testimonials, map embed, and newsletter form

Technologies Used

The project is built with plain HTML for structure, CSS for visual design and responsiveness, and vanilla JavaScript for data rendering and interactions.

Scalability

New events can be added by inserting another object into the shared dataset and creating a matching event HTML file that points to its slug.
